# Matcher

Matchers are field selections

```js
interface Matcher {
  key: string //unique identifer
  operator: string //operator
  comparison: string //comparison
  source: string //field
  value: Value //value
  text: string //text displsay
}
```

### key
Unique key

### operator
Operator

### comparison
Comparison

### source
Field

### value
Value

### text
Text to display
